What is DOM and what is the purpuse the DOM?

DOM full meaning is a Document Object Model.It allows the accessing of the documents like HTML.It allows a platform to be language neutral and provide an interface that allows the dynamic usage of the scripts and programs to access the content.It also provide a way to update the content, structure and style of the document and a way to represent it in any language


How will you select HTML elements using DOM?

There are 5 ways in which we can select elements in a DOM using selectors.


  1. 1. getElementByTagName()
  2. 2. getElementByClassName()
  3. 3. getElementById()
  4. 4. querySelector()
  5. 5. querySelectorAll()

What is event bubble?

The bubbles read only property of the event interface indicates whether the event bubbles up through the DOM tree or not.It is a process that starts with the element that triggered the event and the bubbles up to the containing elements in the hierarchy.In event bubling,the event first captured and handled by the innermost element and then propagated to outer elements.


What is the callback function?

A callback is function passed as an argument to another function.This techniquw allows a function to call another function.A callback function can run after another function has finished

Why use callback function?

Callback functions are needed because many JavaScript actions are asynchronous. Instead, it executes in the background while the rest of the code runs. A callback's purpose is to execute code in response to an event.
